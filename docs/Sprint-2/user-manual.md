# 📘 คู่มือการใช้งานระบบ (User Manual)

## 🎯 Story
As an admin, I want to keep the users updated on their reported incidents.

---

# 🛠 ระบบอัปเดตสถานะและแจ้งความคืบหน้าเหตุการณ์สำหรับผู้ใช้ (PBL6)

ผู้ดูแลระบบสามารถอัปเดตสถานะและความคืบหน้าของเหตุการณ์ที่ผู้ใช้รายงานเข้ามา  
เพื่อแจ้งให้ผู้ใช้ทราบถึงผลการตรวจสอบและการดำเนินการแก้ไขในแต่ละขั้นตอน

ระบบจะแสดงสถานะของเคส เช่น  
**รอตรวจสอบ / ปฏิเสธ / แก้ไขแล้ว**  
พร้อมรายละเอียดการอัปเดต เพื่อให้ผู้ใช้สามารถติดตามข้อมูลได้อย่างต่อเนื่อง

---

# ⭐ ฟีเจอร์หลัก (Features)

- ดูรายการรายงานเหตุการณ์ทั้งหมดจากผู้ขับขี่และผู้โดยสาร
- ตรวจสอบรายละเอียดเคสและหลักฐานที่แนบมา (รูปภาพ/ไฟล์)
- อัปเดตสถานะหรือบันทึกผลการดำเนินการของแต่ละเคส
- ส่งการแจ้งเตือนหรือข้อความแจ้งความคืบหน้าไปยังผู้ใช้ที่เกี่ยวข้อง
- ค้นหา กรอง และติดตามประวัติรายงานย้อนหลังได้อย่างเป็นระบบ

---
