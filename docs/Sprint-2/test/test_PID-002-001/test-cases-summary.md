## รายละเอียดแต่ละ Test Case

### TC-007: แนบไฟล์ที่ไม่รองรับ (.txt) ระบบแสดง Error Dialog
- **เงื่อนไข:** เปิด Report Modal → อัปโหลดไฟล์ `.txt`
- **ขั้นตอน:**
  1. Login → ไปหน้า myTrip → แท็บ "ยืนยันแล้ว" → กดปุ่ม "รายงาน"
  2. แนบไฟล์ `test_invalid.txt`
  3. ตรวจสอบว่า popup "ไฟล์ไม่รองรับ" แสดงขึ้น
  4. กดปุ่ม "ตกลง" → popup หายไป
- **ผลที่คาดหวัง:** แสดง popup error พร้อมชื่อไฟล์ → กดปิดได้

### TC-008: แนบไฟล์รูปภาพที่รองรับ (.jpg) สำเร็จ

### TC-009: เลือก "อื่นๆ" แต่ไม่กรอกรายละเอียด ปุ่มส่งต้อง disabled
- **เงื่อนไข:** เปิด Report Modal → เลือก checkbox "อื่นๆ" → ไม่กรอก textarea
- **ขั้นตอน:**
  1. Login → เปิด Report Modal
  2. เลือก "อื่นๆ" → textarea แสดงขึ้นพร้อมเครื่องหมาย `*` สีแดง
  3. ไม่กรอกอะไร → ตรวจสอบปุ่ม "ส่ง Report"
- **ผลที่คาดหวัง:** ปุ่ม disabled + แสดงข้อความ "กรุณาระบุรายละเอียดเพิ่มเติม"

### TC-011: ปุ่มรายงานเปลี่ยนเป็น "รอดำเนินการ" หลังส่งรายงานสำเร็จ
